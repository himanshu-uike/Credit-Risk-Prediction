# Credit Risk Prediction App

## 📌 Project Overview

This project predicts whether a customer is likely to default on a loan using machine learning. It helps financial institutions assess credit risk and make better lending decisions.

## 🚀 Features

* Predict loan default risk
* Displays probability (risk score)
* Interactive Streamlit app
* End-to-end ML pipeline

## 🧠 Model Used

* XGBoost Classifier
* Threshold optimization for better recall

## 📊 Dataset

* 250,000+ records
* Financial + demographic features

## 💻 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* XGBoost
* Streamlit

## ▶️ Run Locally

```bash
streamlit run app.py
```

## 📷 Output

Predicts:

* Risk Score (%)
* High Risk / Low Risk classification
